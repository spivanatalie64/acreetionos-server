# How to Help

AcreetionOS is a libre, community-driven project. There is no corporate
backing — just people who believe software should be free. Here's how you
can help, regardless of your skill level.

## 🐛 Report Bugs

Found something broken? Open an issue:

- **GitHub**: https://github.com/AcreetionOS-Code/acreetionos/issues
- **GitLab**: https://gitlab.acreetionos.org/natalie/acreetionos-server/-/issues

Include:
- What you were doing
- What you expected to happen
- What actually happened
- Any relevant logs (`journalctl -xe`, `dmesg`, etc.)

## 💡 Suggest Features

Have an idea for how to make AcreetionOS better? Open an issue with the
`enhancement` label. We read every suggestion.

## 🧪 Test Builds

The ISO is built quarterly, but you can build from source anytime:

```bash
git clone https://github.com/AcreetionOS-Code/acreetionos-server.git
cd acreetionos-server
sudo ./build.sh
```

Report any build failures or installation issues.

## 📝 Improve Documentation

Seen a typo? Confusing instructions? Missing information?
Submit a PR to the docs/ directory. Every improvement helps.

## 💻 Contribute Code

- Fork the repo
- Create a branch
- Make your changes
- Submit a pull request

All skill levels welcome. We judge code, not people.

## 🌐 Spread the Word

- Tell a friend about AcreetionOS
- Write a blog post about your experience
- Share on social media with **#KeepLinuxFree** and **#AcreetionOSWillAlwaysBeFree**
- Install it on a spare machine and show someone what libre software looks like

## 💰 Financial Support

AcreetionOS is and always will be free. But infrastructure costs money —
servers, domains, bandwidth, signing keys. If you'd like to help cover costs:

- **GitHub Sponsors**: https://github.com/sponsors/natalie
- **Discord**: https://discord.gg/VHqQkJASw7

## 🤝 Community

Join us on Discord: https://discord.gg/VHqQkJASw7

This is where we discuss development, help each other troubleshoot, and
plan the next release. Everyone is welcome.

---

**Remember: Every contribution matters. A bug report, a pull request, a shared
post — it all adds up. Thank you for being part of this.**
